如何从零开始搭建个人博客 —— Windows版本

本文没有多余的废话，直接步入正题

step 1 注册账号

一个GitHub账号，注册方式请自行搜索

step 2 创建仓库

登录GitHub账号，进入首页；在左侧仓库列表栏中点击New

进入仓库创建页面

  • 在Repository name处填写自己的个人博客仓库名字，须遵循***.github.io**的格式，*****处可填写自己想要的用户名 e.g. zhangsan.github.io
  • Description处填写对仓库的描述
  • 勾选public
  • 最好勾选Add a README file
  • 其余保持默认即可

点击Create repository创建仓库

step 3 为个人博客生成一个页面

进入刚创建的仓库

  1. 点击导航栏处的Setttings选项卡，进入设置界面
  2. 随后点击左侧导航栏的Pages选项，进入页面选项
  3. 之后在Branch区域将分支切换为main (稍后要修改为master)
  4. 点击Save
  5. 之后便可以看到Github为你的博客自动生成了一个url

注：如果没有看到生成的url，可以刷新一下页面或者重新 Settings => Pages进入该界面

访问这个url就可以看到你的个人博客了！内容一般为README的内容~

到这里你就完成了一个基本的个人博客搭建，如果希望自己的博客不只是单纯的文字而是具备一定的~~交互性和美观度~~，可以进行下面内容的配置
